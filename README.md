# FNP-EXCEL-SALES_DSAHBOARD
📊 Sales Performance & Order Analytics Project

This project focuses on analyzing transactional sales data to extract meaningful business insights related to revenue trends, customer purchasing behavior, delivery performance, and occasion-based sales patterns.

The dataset consists of structured order-level records including customer details, product information, pricing (INR), revenue, timestamps (order and delivery), geographic locations, and seasonal occasions. The primary objective of this project is to transform raw operational data into actionable insights using data cleaning, transformation, and analytical techniques.

🔍 Project Objectives

Analyze revenue performance across months, days, and occasions

Identify peak sales hours and high-performing time periods

Evaluate delivery efficiency using order-to-delivery time differences

Examine location-based revenue distribution

Understand customer purchase patterns and product demand

📁 Dataset Attributes

Order ID

Customer ID

Product ID

Quantity Ordered

Product Price (INR)

Revenue

Order Date & Time

Delivery Date & Time

Delivery Time Difference

Location

Occasion

Month Name & Day Name

Order Hour & Delivery Hour

📈 Key Insights Generated

Identified top revenue-generating months and occasions

Determined peak order placement hours

Measured operational efficiency through delivery time analysis

Analyzed regional sales distribution

Evaluated product-level revenue contribution

🛠️ Tools & Techniques Used

Data Cleaning & Preprocessing

Exploratory Data Analysis (EDA)

KPI Calculation

Data Aggregation & Trend Analysis

Dashboard Creation (Excel / Power BI)

🎯 Outcome

The project demonstrates strong analytical and business intelligence capabilities by converting structured transactional data into performance-driven insights that support strategic planning, operational improvements, and revenue optimization.
